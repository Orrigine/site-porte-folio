About the C language

C is a high-level and general-purpose programming language that is ideal for developing firmware or portable applications. Originally intended for writing system software, C was developed at Bell Labs by Dennis Ritchie for the Unix Operating System in the early 1970s.

Our C Project

After learning Git/Github and how works Debian WSL (linus environnement on Microsoft), the aim was to create a Connect-4 with some bonus, if it was possible to do them.
The bonus are a modifiable table to play on more column and row, the detection of the winner and a multiplayer like 6 players at the same time on the same board. The project was a bit hard to realise because understand how works Github was the first difficulty and after the configuration on Debian was hard too.
